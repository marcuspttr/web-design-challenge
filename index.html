Summary: Analysis of Weather Data

The purpose of this project was to analyze weather changes as you approach and gain distance from the equator (a latitude of 0 degrees). To accomplish this analysis, using a dataset of over 500 cities, pulled relevant weather information from the OpenWeatherMap API.

After assembling the dataset, we used Matplotlib to graph various aspects of weather vs. latitude. Analysis was taken further by running linear regressions upon the data. Factors we looked at included: what hemisphere the cities were in, the max temperatures, wind speeds, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
